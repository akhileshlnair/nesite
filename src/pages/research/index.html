DECEMBER 20, 2025 • SYSTEM ARCHITECTURE

Reducing Latency in Local LLM Inference

We have successfully migrated the core Vivral reasoning engine to run entirely on the device's Neural Engine. This eliminates the round-trip time to the cloud, resulting in 0ms network latency.

The challenge was compressing the weights without losing diagnostic accuracy. By using 4-bit quantization specifically tuned for medical terminology, we achieved a 99.8% accuracy retention rate compared to the FP16 model.

NOVEMBER 15, 2025 • PRIVACY PROTOCOLS

The Secure Enclave Approach

Traditional health apps sync to a server. We decided that was unacceptable.

Vivral now operates inside a sandboxed "Secure Enclave" within the OS. This means even if the phone is compromised, the health data remains encrypted at rest using keys that are never stored in memory for more than the duration of a session.
